# Business Sales Performance Analytics

## Project Overview
This project analyzes business sales data to identify revenue trends, top-selling products, profitable categories, and regional sales performance using Python and Jupyter Notebook.

---

## Objective
The objective of this project is to:
- Analyze sales and profit trends
- Identify top-performing products
- Evaluate regional sales performance
- Generate actionable business insights
- Visualize data using charts and graphs

---

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

---

## Dataset Information
Dataset Used: Sample Superstore Dataset

The dataset contains:
- Product details
- Sales information
- Profit values
- Customer orders
- Regional data

Main columns used:
- Sales
- Profit
- Region
- Category
- Product Name
- Order Date

## Key Insights
1. Technology category generated the highest revenue.
2. West region showed the strongest sales performance.
3. Some furniture products generated losses.
4. Sales increased during year-end months.
5. Top products contributed significantly to total sales.
